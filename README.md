## WEB GL SAMPLE
### STEP.1
clone
```
$ git clone https://github.com/y-ono366/web_gl.git
```
### STEP.2
build
```
$ cd web_gl
$ npm install
$ gulp sass
```
### STEP.3
access
```
$ npm start
```
access to http://localhost:8080/
