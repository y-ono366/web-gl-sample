## WEB GL SAMPLE
<img src="https://github.com/y-ono366/web_gl/blob/media/captcha.gif" width="100%"></img>
### STEP.1 CLONE
```
$ git clone https://github.com/y-ono366/web_gl.git
```
### STEP.2 BUILD
```
$ cd web_gl
$ npm install
$ gulp sass
```
### STEP.3 ACCESS
```
$ npm start
```
access to http://localhost:8080/
